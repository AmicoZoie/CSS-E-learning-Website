Formatting Text with CSS

Styling text in CSS refers to how text is displayed and arranged inside an HTML element. By combining font styles and text layout styles, developers improve readability, visual appeal, and design consistency.

1. Font Properties

• Font-Family: Specifies the typeface. A Font Stack (a list of fonts) ensures text displays properly if the preferred font is missing.

• Font-Size: Controlled via px (static), %, or em (dynamic/proportional).

• Font-Style: Options include normal, italic (cursive-style), or oblique (slanted).

• Font-Weight: Controls thickness, ranging from 100 to 900 or keywords like bold.

• Font-Variant: Used to create small-caps effects.

2. Text Layout & Alignment

• Text-Align: Options include left, right, center, and justify (common in formal publications).

• Text-Decoration: Adds an underline, overline, or line-through.

• Text-Transform: Changes case to uppercase, lowercase, or capitalize.

• Line-Height: Sets vertical distance; extra space created here is called leading.

• Spacing: letter-spacing and word-spacing adjust the gaps between characters and words.

3. Borders, Padding & Lists

Borders: Defined by style (dotted, solid, double, etc.), width, and color. You can use border-radius to round the corners of an element.

Padding vs. Margin: Padding is "filling" inside an element (between content and border), while Margin is the space outside the element.

Lists: list-style-type changes bullets (disc, circle, square) or numbering (decimal, roman, alpha).

4. Pseudo-Classes & Elements

Pseudo-classes (:) style dynamic states, like :hover when a mouse is over an element or :focus when selected.

Pseudo-elements (::) style specific parts, like ::first-letter for drop caps or ::before/::after to insert generated content.
