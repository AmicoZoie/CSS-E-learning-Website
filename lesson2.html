HTML Frames

Frames divide the browser window into multiple sections, each displaying a different HTML document.


Frameset

• <frameset> replaces the <body> tag

• rows – divides horizontally

• cols – divides vertically



Frame Tag

• src – specifies which page to load

• name – names the frame

• noresize – prevents resizing


Frames are mostly outdated today and replaced by CSS layouts.
