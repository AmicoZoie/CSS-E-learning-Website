HYPERLINKS

In this lesson, you will learn how hyperlinks work in HTML. Hyperlinks allow users to move from one page to another, connect to other websites, and navigate within the same page.


What You Will Learn

• What is the anchor tag (<a>)

• Absolute and Relative Links

• Opening links in a new tab

• Linking images and text



Example Code

<a href="https://www.example.com">Visit Website</a>


After this lesson, you should be able to create hyperlinks and connect different web pages together.
