Formatting Text with CSS

Styling text in CSS refers to how text is displayed and arranged inside an HTML element[cite: 4]. By combining font styles and text layout styles, CSS improves readability, visual appeal, and design consistency[cite: 12].


Core CSS Properties

• Font Styles: Control appearance like font-family, size, weight (bold), and style (italic)[cite: 10].

• Text Layout Styles: Affect spacing and positioning, including line spacing, letter spacing, and alignment[cite: 11].

• Text-Decoration: Adds or removes lines like underline, overline, and line-through[cite: 67].

• Text-Transform: Changes text case to uppercase, lowercase, or capitalize without altering original content[cite: 72, 73].



Important Concepts

The Font Stack: A list of fonts in the font-family property that allows the browser to choose an alternative if the preferred one is unavailable[cite: 28].


Line Height & Leading: The line-height property sets the vertical distance between lines[cite: 94]. Extra space created when line-height is larger than font size is called leading[cite: 98].


Example CSS Code

p {
  font-family: "Inter", sans-serif; /* Font Stack [cite: 28] */
  font-size: 16px; /* Font Size [cite: 34] */
  color: #ff0000; /* Hex-notation [cite: 20] */
  text-align: justify; /* Horizontal Alignment [cite: 60] */
}
