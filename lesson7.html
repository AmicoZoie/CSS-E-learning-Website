Backgrounds & The Box Model

In this lesson, you will learn how to style backgrounds and understand how spacing works in CSS using the Box Model.


Background Properties

• background-image: Adds an image behind an element.

• background-color: Sets a solid color background.

• background-repeat: Controls if the image repeats (repeat, no-repeat, repeat-x, repeat-y).

• background-position: Sets where the image appears (center, top, left, right, bottom).



Offset & Spacing

• Margin: Space outside the element (pushes elements apart).

• Padding: Space inside the element (between content and border).



The CSS Box Model

Every HTML element is treated as a rectangular box. The Box Model consists of:

• Content – The actual text or image

• Padding – Space around the content

• Border – The edge around padding

• Margin – Space outside the border



Example CSS Code

div {
  background-color: black;
  background-image: url("bg.jpg");
  background-repeat: no-repeat;
  background-position: center;
  padding: 20px;
  margin: 15px;
}



Activity

Create a div element and:

1. Add a background color.

2. Insert a background image.

3. Set it to no-repeat.

4. Center the image.

5. Add padding and margin.

6. Add a visible border to see the Box Model clearly.


Observe how spacing changes when you adjust margin and padding.
