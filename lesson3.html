Web Forms

Web forms collect information from users. They are used for registration, surveys, login pages, and feedback.


Form Structure

• <form> tag contains all input elements

• Action – where the data is sent

• Method – how the data is sent (GET or POST)



Common Input Types

• Text – single-line input

• Password – masked characters

• Email – accepts valid email format

• Number – numeric input only

• Radio – choose ONE option

• Checkbox – choose MULTIPLE options



Advanced Form Elements

• Dropdown – select and option

• Textarea – multi-line input

• Submit – sends data

• Reset – clears form
