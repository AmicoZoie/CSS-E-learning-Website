Introduction to CSS

Cascading Style Sheets (CSS) is a language used to describe the presentation of a web page including colors, layout, and fonts.


Why CSS is Important

• Controls colors, layout, spacing, and design

• Adapts websites to different devices (desktop, mobile, printer)

• Works independently from HTML

• Allows one CSS file to control multiple web pages



History of CSS

CSS was developed by the W3C because HTML became too complex when styling tags like <font> were added.

CSS separated design from content — HTML handles structure, CSS handles appearance.



How CSS Works

Selector: Targets the element (example: p)

Property: What you want to change (example: color)

Value: The setting applied (example: blue)


Example:

p { color: red; text-align: center; }
