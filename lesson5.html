CSS Selectors, Div, Span & Links

In this lesson, you will learn how CSS selects elements and how div, span, and links are used to structure and style web pages.


CSS Selectors

• Element Selector: Selects all elements of a type (example: p, h1).

• Class Selector: Selects elements with a specific class (example: .box).

• ID Selector: Selects one unique element (example: #header).

• Universal Selector: Selects all elements (*).



Div and Span

• Div: A block-level container used to group large sections of content.

• Span: An inline container used to style small parts of text inside elements.

A div usually takes the full width of the page, while a span only takes as much space as its content.



Styling Links

• a: Styles normal links.

• a:hover: Changes style when the mouse is over the link.

• a:visited: Styles visited links.

• text-decoration: Removes or adds underline.



Example CSS Code

.box {
  background-color: black;
  color: white;
}

#title {
  color: red;
}

a:hover {
  color: #e50914;
}



Activity

Create a simple webpage and:

1. Add a div that contains a heading and a paragraph.

2. Use a class selector to change its background color.

3. Add a span inside the paragraph to highlight one word.

4. Add a link and apply a hover effect.

5. Use an ID selector to change the heading color.


Observe how selectors allow you to target specific elements and control your page design precisely.
