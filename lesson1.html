Hyperlinks

Hyperlinks connect web pages together. They are created using the <a> (anchor) tag.


Basic Structure

<a href="page.html">Click Here</a>


Types of Links

• Absolute – full website URL

• Relative – link within same folder

• Email link – opens mail application

• Target attribute – opens in new tab
